## Please read this carefully before starting to contribute:

1. We do not accept contributions from non-organization members. If you are excited about the project, please consider joining in.
2. Rules are found in the main repository `documentation` folder.
3. Read [project rules](documentation/rules/project-rules.md) before adding or editing issues or milestones.
4. Read [git rules](documentation/rules/git-flow.md) before commiting code.
